
Explicit memory: Memory in which there is a need for conscious recollection in order to recall something. By contrast, in <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=33196" onclick="wmdTrack('embd-lnk');" rel="dict">implicit memory</a> there is a lack of conscious awareness in the act of recollection. Implicit memory may survive largely unimpaired at the same time as a person's powers of explicit memory decline with age or are devastated in Alzheimer disease.
