
Accessory: Additional, extra, supplementary, subsidiary to the main thing. An <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=25962" onclick="wmdTrack('embd-lnk');" rel="dict">accessory digestive organ</a> is an organ that helps with digestion but is not part of the digestive tract.
The <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=7645" onclick="wmdTrack('embd-lnk');" rel="dict">accessory nerve</a> is so-called  because it receives an additional (accessory) root from the upper part of the spinal cord.
An <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=2104" onclick="wmdTrack('embd-lnk');" rel="dict">accessory placenta</a> is an extra placenta separate from the main placenta.
