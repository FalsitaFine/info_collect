
Chromatin remodeling: Dynamic structural changes to the chromatin occurring throughout the cell division cycle. These changes range from the local changes necessary for transcriptional regulation to global changes necessary for chromosome segregation. Chromatin remodeling is an <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=21819">epigenetic</a> phenomenon.
