
Gingko biloba: An herb that is very popular as a treatment for <a href="https://www.medicinenet.com/dementia/article.htm" onclick="wmdTrack('embd-lnk');" rel="dt">dementia</a> (a progressive brain dysfunction) and to improve thinking. Research demonstrates some support for the use of gingko for the following conditions: dementia and <a href="https://www.medicinenet.com/alzheimers_disease_causes_stages_and_symptoms/article.htm" onclick="wmdTrack('embd-lnk');" rel="dt">Alzheimer's disease</a>, intermittent <a href="https://www.medicinenet.com/claudication/article.htm" onclick="wmdTrack('embd-lnk');" rel="dt">claudication</a>, <a href="https://www.medicinenet.com/glaucoma/article.htm" onclick="wmdTrack('embd-lnk');" rel="dt">glaucoma</a>, memory enhancement, <a href="https://www.medicinenet.com/macular_degeneration/article.htm" onclick="wmdTrack('embd-lnk');" rel="dt">macular degeneration</a>, <a href="https://www.medicinenet.com/tinnitus_ringing_in_the_ears/article.htm" onclick="wmdTrack('embd-lnk');" rel="dt">tinnitus</a>, and <a href="https://www.medicinenet.com/raynauds_phenomenon/article.htm" onclick="wmdTrack('embd-lnk');" rel="dt">Raynaud's phenomenon</a>.   Gingko can cause mild stomach upset and <a href="https://www.medicinenet.com/headache/article.htm" onclick="wmdTrack('embd-lnk');" rel="dt">headache</a> and does interact with other non-prescription and prescription medications.     Gingko seems also to have blood thinning properties.  Therefore, it is not recommended to be taken with aspirin, nonsteroidal anti-inflammatory drugs (such as <a href="https://www.medicinenet.com/ibuprofen/article.htm" onclick="wmdTrack('embd-lnk');" rel="pharm">ibuprofen</a>/Advil, Motrin or <a href="https://www.medicinenet.com/naproxen/article.htm" onclick="wmdTrack('embd-lnk');" rel="pharm">naproxen</a>/Aleve), or medications to prevent blood clotting (anticoagulants) such as <a href="https://www.medicinenet.com/warfarin/article.htm" onclick="wmdTrack('embd-lnk');" rel="pharm">warfarin</a>/Coumadin. Gingko should be avoided in patients with epilepsy taking <a href="https://www.medicinenet.com/seizure/article.htm" onclick="wmdTrack('embd-lnk');" rel="dt">seizure</a> medicines, such as <a href="https://www.medicinenet.com/phenytoin/article.htm" onclick="wmdTrack('embd-lnk');" rel="pharm">phenytoin</a> /Dilantin, <a href="https://www.medicinenet.com/carbamazepine/article.htm" onclick="wmdTrack('embd-lnk');" rel="pharm">carbamazepine</a>/Tegretol, and phenobarbital.  Also, <a href="https://www.medicinenet.com/antidepressants/article.htm" onclick="wmdTrack('embd-lnk');" rel="pharm">antidepressants</a>, medications for hypertension, medications to lower blood sugar, cyclosporine, and thiazide diuretics.
