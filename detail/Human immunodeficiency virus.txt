
<a href="https://www.medicinenet.com/script/main/art.asp?articlekey=3812" onclick="wmdTrack('embd-lnk');" rel="dict">Human immunodeficiency virus</a>: <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=3769" onclick="wmdTrack('embd-lnk');" rel="dict">HIV</a>, the cause of <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=2181" onclick="wmdTrack('embd-lnk');" rel="dict">AIDS</a>. HIV has also previously been called the human lymphotropic <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=5997" onclick="wmdTrack('embd-lnk');" rel="dict">virus</a> type III, the <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=6473" onclick="wmdTrack('embd-lnk');" rel="dict">lymphadenopathy-associated virus</a> and the <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=6472" onclick="wmdTrack('embd-lnk');" rel="dict">lymphadenopathy virus</a>. No matter what name is applied, it is a <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=5344" onclick="wmdTrack('embd-lnk');" rel="dict">retrovirus</a>. (A retrovirus has an <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=5382" onclick="wmdTrack('embd-lnk');" rel="dict">RNA</a> <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=3580" onclick="wmdTrack('embd-lnk');" rel="dict">genome</a> and a <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=5347" onclick="wmdTrack('embd-lnk');" rel="dict">reverse transcriptase</a> <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=3266" onclick="wmdTrack('embd-lnk');" rel="dict">enzyme</a>. Using the reverse transcriptase, the virus uses its RNA as a template for making complementary <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=13963" onclick="wmdTrack('embd-lnk');" rel="dict">DNA</a> which can integrate into the DNA of the <a href="https://www.medicinenet.comhttps://www.medicinenet.com/liver_anatomy_and_function/article.htm03" onclick="wmdTrack('embd-lnk');" rel="dict">host</a> organism). Although the American researcher Robert Gallo at the <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=9786" onclick="wmdTrack('embd-lnk');" rel="dict">National Institutes of Health</a> believed he was the first to find HIV, it is now generally accepted that the French <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=4887" onclick="wmdTrack('embd-lnk');" rel="dict">physician</a> Luc Montagnier (1932-) and <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=22858" onclick="wmdTrack('embd-lnk');" rel="dict">his</a> team at the <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=24056" onclick="wmdTrack('embd-lnk');" rel="dict">Pasteur Institute</a> discovered HIV <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=3950" onclick="wmdTrack('embd-lnk');" rel="dict">in</a> 1983-84.
