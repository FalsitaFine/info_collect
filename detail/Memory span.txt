
Memory span: The number of items, usually words or numbers, that a person can retain and recall. Memory span is a test of <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=7143" onclick="wmdTrack('embd-lnk');" rel="dict">working memory</a> (<a href="https://www.medicinenet.com/script/main/art.asp?articlekey=7142" onclick="wmdTrack('embd-lnk');" rel="dict">short-term memory</a>). In a typical test of memory span, an examiner reads a list of <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=39535" onclick="wmdTrack('embd-lnk');" rel="dict">random numbers</a> aloud at about the rate of one number per second. At the end of a sequence, the person being tested is asked to recall the items, in order. The average memory span for normal adults is seven.
