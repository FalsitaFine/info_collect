
Granuloma, calcified: A granuloma that contains calcium deposits. Because it usually takes time for calcium to be deposited in a granuloma, a <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=8908" onclick="wmdTrack('embd-lnk');" rel="dict">calcified granuloma</a> is generally assumed to be an old granuloma.
