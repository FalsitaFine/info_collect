
<a href="https://www.medicinenet.com/script/main/art.asp?articlekey=11328" onclick="wmdTrack('embd-lnk');" rel="dict">Enterostomy</a>: A surgical <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=10115" onclick="wmdTrack('embd-lnk');" rel="dict">operation</a> that opens the <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=5512" onclick="wmdTrack('embd-lnk');" rel="dict">small intestine</a> and brings it through the <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=19269" onclick="wmdTrack('embd-lnk');" rel="dict">abdominal</a> wall to create a new opening (a <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=5559" onclick="wmdTrack('embd-lnk');" rel="dict">stoma</a>) to permit intestinal draining. Like a <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=2796" onclick="wmdTrack('embd-lnk');" rel="dict">colostomy</a>, an enterostomy is a type of <a href="https://www.medicinenet.com/script/main/art.asp?articlekey=4689" onclick="wmdTrack('embd-lnk');" rel="dict">ostomy</a>.
